# Tic Tac Toe Game in C

A console-based Tic Tac Toe game developed in C.

## Features

* Play on 3x3, 4x4, or 5x5 boards
* Choose your symbol (X or O)
* Automatic win and tie detection
* Move history logging to `moves.txt`
* Timestamp recording for every move
* Dynamic memory allocation
* Special 3x3 mode where only the latest 3 moves of each player remain on the board

## How to Compile

```bash
gcc main.c -o main
```

## How to Run

```bash
./main
```

On Windows:

```bash
main.exe
```

## How to Play

1. Choose your symbol (X or O)
2. Select a board size (3, 4, or 5)
3. Enter row and column coordinates for each move
4. Continue until a player wins or the game ends in a tie

## Files

* `main.c` – source code
* `moves.txt` – stores move history and game results
